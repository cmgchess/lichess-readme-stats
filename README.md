## Parameters

| Parameter  | Type   | Description                            | Default      |
| ---------- | ------ | -------------------------------------- | ------------ |
| `username` | string | Your Lichess username                  | **Required** |
| `theme`    | string | Theme for the card (`light` or `dark`) | `dark`      |
